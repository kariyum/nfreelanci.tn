# nfreelanci.tn

nfreelanci.tn is a freelancing platform for freelancers to connect!

The platform aims to solve basic needs, allowing users to interact seamlessly, work and track their project's progress. 
The application is built with modern best practices to ensure reliability and ease of future improvements.

## Key Features

- Secure registration and login
- Real-time chat and updates
- Notifications
